# MASTERY

Track your path to **10,000 hours**. Log minutes, build counters, use mastery templates.

## Features

- **10,000 hour mastery templates** — psychotherapy, writing, instrument, programming, and more
- Multiple counters with progress bars
- Minute quick-add (15m, 20m, 30m, 45m) + custom duration
- Daily habit templates
- Guest mode or Google sync
- Installable PWA

## Quick start

```bash
npm install && npm run dev
```

Open [http://localhost:3000](http://localhost:3000) — try **Templates** for mastery examples.

## Production

```bash
docker compose up -d --build
```

See `.env.example` for optional Google OAuth.
